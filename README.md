# blinkit_grocery_store_Data_Analysis
End-to-end data analysis project on BlinkIT grocery dataset using Python, Pandas, SQL, PowerBi and data visualization to generate business insights and build a scalable analytics workflow
The goal is to understand sales patterns, product performance, and outlet efficiency to support data-driven decision making.

##  Tech Stack
- Python 
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL (SQLite)
- Jupyter Notebook
- PowerBi

##  Data Cleaning Steps
- Handled missing values
- Removed duplicates
- Standardized column names
- Fixed data types

---

##  Exploratory Data Analysis (EDA)
- Total sales analysis
- Category-wise sales
- Outlet performance
- Price distribution
- Correlation analysis

---

## SQL Key Insights
- Total Sales
- Sales by Item Type
- Top 5 Selling Products
- Sales by Outlet Type
- Average Sales per category
- High Value Product
- Contribution Percent
- Which Outlet is best?
- Which Category has Highest Sales?

##  Visualizations
- Bar charts for category & outlet performance
- Histogram for sales distribution
- Heatmap for correlation analysis
